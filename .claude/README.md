# .claude

Claude Code configuration for the Synchestra repository.

## Contents

- **`settings.json`** — shared Claude Code settings for this repository (committed).
- **`settings.local.json`** — local overrides (not committed).
- **`skills/`** — symlink to [`../ai-plugin/skills/`](../ai-plugin/skills/README.md). Claude Code auto-discovers skills by scanning this directory for `SKILL.md` files.
- **`worktrees/`** — scratch space for Claude Code worktree checkouts.
