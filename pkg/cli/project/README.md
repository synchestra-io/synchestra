# project

Implements the `synchestra project` command group. Currently contains the `new` subcommand which creates a Synchestra project by linking spec, state, and code repos.

Includes config file types and read/write helpers for `synchestra-spec.yaml`, `synchestra-state.yaml`, and `synchestra-code.yaml`, as well as project title derivation logic.
