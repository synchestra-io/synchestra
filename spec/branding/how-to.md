# How to develop characters and website design

For brainstorming illustration visual style (color theme, characters, setting), the most relevant agents are:

1. **Brand Guardian** — Best starting point. Defines brand identity, visual positioning, and ensures consistency. Will help establish the foundational visual language.
2. **Visual Storyteller** — Great for the narrative side: how characters, settings, and scenes tell your brand's story through illustrations.
3. **Image Prompt Engineer** — Once the style is defined, this agent translates your visual concepts into precise prompts for AI image generation tools.
4. **Inclusive Visuals Specialist** — Important if your illustrations include people/characters. Ensures culturally accurate, non-stereotypical representation.
5. **UI Designer** — For integrating the illustration style into your actual website design system (color palettes, component usage).

**Recommended flow:**

- Start with Brand Guardian to establish identity and visual direction
- Then Visual Storyteller to develop character/setting concepts
- Then Image Prompt Engineer to create actionable generation prompts
- Layer in Inclusive Visuals Specialist if characters depict diverse people
