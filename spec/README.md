# Specifications

Technical specifications for Synchestra's development — features, architecture, and design documents.

These are internal artifacts for developers and AI agents building the product. For user-facing documentation, see [`docs/`](../docs/README.md).

## Contents

| Directory | Description |
|---|---|
| [project-definition/](project-definition/README.md) | Project file format and repository layouts |
| [features/](features/README.md) | Feature specifications |

### project-definition

Defines the `synchestra-project.yaml` format, mandatory and optional fields, and the two supported repository layouts: multi-project (under `synchestra/projects/`) and dedicated project repos (project files at root).

### features

Feature specifications for Synchestra's planned capabilities — micro-tasks, cross-repo sync, model selection, and more.
