# `task claim` Arguments

**Parent:** [claim](../README.md)

Arguments specific to the `synchestra task claim` command.

## Arguments

| Argument | Type | Required | Description |
|---|---|---|---|
| [`--run`](run.md) | String | Yes | Unique identifier for this agent run |
| [`--model`](model.md) | String | Yes | Model being used for this work |

Also uses [`--project`](../../../_args/project.md) and [`--task`](../../_args/task.md).

### `--run`

Identifies the agent run claiming the task. See [run.md](run.md).

### `--model`

Records which model the agent is using. See [model.md](model.md).
