# `task complete` Arguments

**Parent:** [complete](../README.md)

Arguments specific to the `synchestra task complete` command.

## Arguments

| Argument | Type | Required | Description |
|---|---|---|---|
| [`--summary`](summary.md) | String | No | Brief description of what was accomplished |

Also uses [`--project`](../../../_args/project.md) and [`--task`](../../_args/task.md).

### `--summary`

Records what was accomplished when completing a task. See [summary.md](summary.md).
