# `task list` Arguments

**Parent:** [list](../README.md)

Arguments specific to the `synchestra task list` command.

## Arguments

| Argument | Type | Required | Description |
|---|---|---|---|
| [`--status`](status.md) | String | No | Filter tasks by status |
| [`--fields`](fields.md) | String | No | Select specific output fields |

Also uses [`--project`](../../../_args/project.md) and [`--format`](../../_args/format.md).

### `--status`

Filters the task list to a specific status. See [status.md](status.md).

### `--fields`

Controls which fields appear in the output. See [fields.md](fields.md).
