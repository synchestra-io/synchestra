# Config Arguments

**Parent:** [config](../README.md)

Arguments shared across `synchestra config` subcommands.

## Arguments

| Argument | Type | Required | Supported by |
|---|---|---|---|
| [`--repos-dir`](repos-dir.md) | String (directory path) | No | `set`, `clear` |

### `--repos-dir`

The root directory where repositories are stored on disk. See [repos-dir.md](repos-dir.md).
