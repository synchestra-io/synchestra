# `project new` Arguments

**Parent:** [new](../README.md)

Arguments specific to `synchestra project new`.

## Arguments

| Argument | Type | Required | Description |
|---|---|---|---|
| [`--title`](title.md) | String | No | Project title |

Also uses [`--spec-repo`](../../_args/spec-repo.md), [`--state-repo`](../../_args/state-repo.md), and [`--code-repo`](../../_args/code-repo.md).

### `--title`

Human-readable project title. If omitted, derived from the spec repo's `README.md` heading or the repo identifier. See [title.md](title.md).
